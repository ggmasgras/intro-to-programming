HTML

- Hypertext Markup Language -- basis of almost every web page

- HTML glues everything together; HTML documents are the "heart" of the web

Major Pieces of the Web

- you & your computer + browser (IE, Chrome, Firefox, Safari, etc.)

- The Internet

- Servers

Your computer browser makes a request over the internet to the servers using the HTTP protocol, and the servers respond back via the internet.

Servers are like computers, just optimized for sitting in a closet, hosting files.
HTTP is a simple protocol that the browser uses to communicate with the servers.

The internet uses HTTO to communicate with you, your browser and servers.

HTML is made up of:

- text content -- what you see

- markup -- what is looks like

- references to other documents -- e.g. images & video

- links to other pages

HTML Markup

HTML Attributes

Anchors - are used for creating links. They include:

- href - attribute name
- link (i.e. www.reddit.com )- the value

Image tag includes:

- img opening/closing tags
- source (src)
- alt (alternate) - text that displays when the image doesn't load -- REQUIRED

Image tags are called "void" tags; they have no content and thus require no closing tag.

Break is also a void tag, used for line breaks.

Paragraph tag is NOT a void tag; it creates a box around the content so it needs a closing tag.

The break tag is an inline tag; it is ending a line ("the line ends here").

The paragraph p tag is a block; it creates an 'invisible' box around the text. This box can have height and width--which can be customized, while the break tag only ends the line.

Other inline elements are bold tag, emphasis tag, and the image tag.

Boxes & Containers -- Span & Div Elements

Span is an inline element creating an inline container element.
Div is a block element creating a 'box' container element. These elements do not do anything other than contain their content. There is a way to attach styling elements to them to influence the way in which they are displaying.

HTML Document

Includes:
- doctype
- HTML tag - encompasses all the content on the page
- head tag- includes info such as metadata, Java script, CSS, as well as the title tag with the title of the page
- body tag - includes the content of the document
